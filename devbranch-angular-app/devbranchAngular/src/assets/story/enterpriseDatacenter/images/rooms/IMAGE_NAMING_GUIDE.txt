IMAGE NAMING GUIDE FOR GAME AUTO-LOADING

Base folder:
story/enterpriseDatacenter/images/rooms/

Each room folder must contain:
1) room image:
   room.webp
2) clue images folder:
   clues/
3) scene images folder:
   scenes/

Rules:
- Room image filename is always exactly: room.webp
- Clue image filename is always exactly: <clue_id>.webp
- Scene image filename is always exactly: <scene_id>.webp
- Use lowercase ids exactly as in storyPlot1.json
- Recommended format: webp

Implemented fallback support in game:
- If .webp is missing, the game tries .png, then .jpg, then .jpeg.
- This applies to room images, clue images, and scene images.
